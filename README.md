# Laser Gun (NodeMCU ESP8266 + CD4026)

Simple laser gun project with:
- trigger & reload buttons
- laser module
- green / red LEDs
- passive buzzer (sound effects)
- 7-segment ammo counter via CD4026

---

## Power
- NodeMCU logic level: **3.3V**
- CD4026 powered from **3.3V**
- **Common GND** for all components
- Each 7-seg segment must have its **own 220–330Ω resistor**

---

## Pin Mapping (NodeMCU Lua IO index)

| Function        | IO | NodeMCU | GPIO |
|-----------------|----|---------|------|
| Trigger button  | 1  | D1      | 5    |
| Reload button   | 2  | D2      | 4    |
| Laser           | 5  | D5      | 14   |
| Green LED       | 6  | D6      | 12   |
| Red LED         | 7  | D7      | 13   |
| Buzzer          | 0  | D0      | 16   |
| CD4026 CLK      | 3  | D3      | 0    |
| CD4026 RESET    | 4  | D4      | 2    |

---

## Buttons
Configured with internal pull-ups.

- One side of button → **GND**
- Other side → pin (D1 or D2)
- Pressed = LOW

---

## LEDs
Use series resistor **220–330Ω**.

- Anode → pin (D6 / D7)
- Cathode → GND

---

## Buzzer
Passive buzzer only.

- One pin → **D0**
- Other pin → **GND**

Sound is generated by software pin toggling.

---

## Laser Module
For low-power 3.3V laser modules:

- `+` → **D5**
- `-` → **GND**

For higher power modules, use a transistor or MOSFET (not covered here).

---

## CD4026 + 7-Segment Display

### CD4026 control
- **VDD** → 3.3V
- **GND** → GND
- **CLK** → D3
- **RESET** → D4
- Clock Inhibit → GND
- Display Enable In → 3.3V

### 7-segment
- CD4026 segment outputs → **resistor** → segment pins
- One resistor per segment (a–g)

---

## Logic Summary
- Trigger: fire laser, play sound, decrement ammo
- Empty magazine: error sound + red LED
- Reload:
  - Blocked if magazine is full (sound + blink)
  - Otherwise reload animation, refill ammo, confirmation sound
